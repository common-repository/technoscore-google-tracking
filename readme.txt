== Description ==

Technoscore Google Tracking is  best Google Analytics plugin for WordPress. See how visitors find and use your website, so you can keep them coming back.

= More details are as follows:- =
* Technoscore Google Tracking adds Google AdWords  Conversion tracking , Remarketing Tag  and Google Analytics  to header/footer part of Selected webpages.
* Technoscore Google Tracking allows you to track your blog visitors easily.
* Really easy way to insert Google Analytics tracking code and keep it there when switching themes. No need to copy/paste code anywhere. This is the best way to handle Google Analytics, Remarketing  in WordPress.

== Installation ==

1. Use WordPress' built-in installer
2. Download the ZIP file from WordPress website
3. Extract the technoscore-google-tracking directory to your computer
4. Upload the technoscore-google-tracking directory to the /wp-content/plugins/ directory
5. Activate the plugin in the Plugin dashboard
